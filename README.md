# Sistema Master - Back-end

Este é o servidor do projeto de Gestão de Estoque e Serviços, desenvolvido em Node.js.

## 🚀 Tecnologias
- Node.js
- Express
- (Adicione aqui seu banco de dados, ex: PostgreSQL ou MySQL)

## 📦 Como rodar
1. Clone o repositório:
   `git clone https://github.com/lucasribeironicoleti/sistema_master-back.git`
2. Instale as dependências:
   `npm install`
3. Crie um arquivo `.env` na raiz seguindo o `.env.example`.
4. Inicie o servidor:
   `npm run dev`

## 🛠 Endpoints Principais
- `GET /servicos` - Lista todos os serviços
- `POST /servicos` - Cadastra um novo serviço
- `GET /produtos` - Lista todos os produtos
